Simple LCD Test

The easiest way to locate bad or stuck pixels on an LCD screen is to display a solid color across the entire display.

This test allows you to perform a full screen test using base colors: white, red, green, blue, yellow, cyan, magenta and black.

Use fullscreen mode in your browser (F11 key) to maximaze page view. Click mouse button anywhere on the page to change color.
